# mvvm_skill_test

一个用于验证 `flutter-mvvm-template` 的 Flutter 示例项目。

## 开始使用

这个项目由模板生成，用于检查 MVVM 基类、sealed page 导航和通用弹窗 UI 是否正常工作。

如果你是第一次使用 Flutter，可以参考：

- [Lab: Write your first Flutter app](https://docs.flutter.dev/get-started/codelab)
- [Cookbook: Useful Flutter samples](https://docs.flutter.dev/cookbook)

更多 Flutter 开发资料可以查看 [online documentation](https://docs.flutter.dev/)，里面包含教程、示例、移动端开发指南和完整 API 参考。
